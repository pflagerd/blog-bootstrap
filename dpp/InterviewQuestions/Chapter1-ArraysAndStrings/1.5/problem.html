Problem Statement

There are three types of edits that can be performed on strings: insert a character, remove a character, or replace a character. Given two strings, write a function to check if they are one edit (or zero edits) away.

Constraints

0 ≤ len(string) ≤ 105

Expected Complexity

Time: O(n)
Space:O(1)

Example 1

Removing the "a" is a single edit
string = pale, ple
expectedResult = True

Example 2

Removing the "s" is a single edit
string = pales, pale
expectedResult = True

Example 3

Replacing the "p" with the "b" is a single edit
string = pale, bale
expectedResult = True

Example 4

Replacing the "p" with the "b" is one edit, replacing the "l" with the "k" is another edit
string = pale, bake
expectedResult = False

Entities and their definitions

string: A sequence of zero or more characters.

empty string: A sequence of zero characters.

Analysis

Since a string is a sequence of characters, the order of the characters is important

Since the only operations allowed on strings are to 1) insert a character, 2) remove a character, or 3) replace a character, the length of the two strings can differ at most by one character.

If the strings have the same length (their lengths differ by 0 characters), the only possible way to transform one into the other is by changing exactly one character.

If the strings' lengths differ by 1 character, there are four ways to transform one into the other

If the first string is longer than the second string, ...

...we can insert one character into the second string.

OR ...we can delete one character from the first string.

If the first string is shorter than the second string, ...

...we can insert one character into the first string.

OR ...we can delete one character from the second string.
