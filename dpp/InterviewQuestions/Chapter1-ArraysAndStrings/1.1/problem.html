Problem Statement

Implement an algorithm to determine if a string has all unique characters. What if you cannot use additional data structures?

Constraints

0 ≤ len(string) ≤ 105

Expected Complexity

Time: O(n)
Space:O(n)

Example 1

There are are no characters in the string, so it's impossible to answer the question.
string = 
expectedResult = Indeterminate

Example 2

There is one character in the string.
string = a
expectedResult = True

Example 3

Both characters are different
string = ab
expectedResult = True

Example 4

Both characters are the same
string = aa
expectedResult = False

Example 5

t, h, e, u, r, o, ' ' are duplicated.
string = the quick brown fox jumps over the lazy dog
expectedResult = False
