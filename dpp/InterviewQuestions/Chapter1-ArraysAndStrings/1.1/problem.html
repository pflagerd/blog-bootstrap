The problem statement is:
Given heights h[] of N towers, the task is to bring every tower to the same height by either adding or removing blocks in a tower. Every addition or removal operation costs cost[] a particular value for the respective tower. Find out the Minimum cost to Equalize the Towers.

Expressing this in mathy notation:

Let cost[] be rewritten as \(c_i\)
Let h[] be rewritten as \(h_i\)
$$i, c, c_i, h, h_i, N \in \mathbb{N} \tag{1.0}$$
\(i, c, c_i, h, h_i, N\) are natural (non-negative) numbers



Let \(c(h)\) represent the cost of using a given value of \(h\) to equalize the towers, thus:
$$c(h) = \sum\limits_{i=0}^{N - 1} |h_i - h| \times c_i \tag{2.0}$$



Let \(c_i(h)\) represent the cost for one particular tower as a function of \(h\), thus:
$$C_i(h) = |h_i - h| \times c_i \tag{3.0}$$



Does \(c(h)\) have a unique minimum value for all \(h_i, c_i, N\) ?
