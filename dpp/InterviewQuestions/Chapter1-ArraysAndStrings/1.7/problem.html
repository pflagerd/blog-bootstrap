Problem Statement

Given an image represented by an NxN matrix, where each pixel in the image is 4 bytes, write a method to rotate the image by 90 degrees. Can you do this in place?

Expected Complexity

Time: O(n)
Space: O(1)

Example 1

repeated characters replaced with a number.
string = aabcccccaaa
expectedResult = a2b1c5a3

Example 2

Cannot compress it more. Return the same string as original.
string = abcdef
expectedResult = abcdef
