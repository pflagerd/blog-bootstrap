🧠 Analysis & Ontology

  • Construct the ontology.
  • Identify the "Features":
  • Objects
  • Constraints
  • Actions (Functions)
  • Can the problem (as opposed to the solution) be expressed symbolically (mathematically)?
  • State the ontology in the Analysis.
  • What kind of reference machine is needed to solve the problem? (e.g. state machine, pushdown automaton, etc.)

💻 Implementation

  • Implicit mental task: Look closely at the given example for any constraints or gotchas.
  • Code the problem using the appropriate automata.
  • Code the problem a different (organic) way.
  • Optimize the solution until it is the best it can be for space and time.
